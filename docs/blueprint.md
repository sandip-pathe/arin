# **App Name**: Anaya

## Core Features:

- Sidebar Navigation: Left sidebar for easy navigation. Includes: Logo/Branding, New Chat, Search Chat, Library, Tools, Chat/Document History List, Settings, Account/Profile
- Primary Interaction Area (Center Pane): Multi-modal input box to paste text, upload files (PDF, Word, Images), add links, and capture images/documents.
- Action Options Below Input: Action options below input: Summarize, Info Extract, Visualize, Jurisdiction Selector (Indian Law, US Law, EU Law, Trade Law), Response Type Selector (Fast Answer, Slow Thinking, Auto).
- Output & Follow-up Section: Output appears in a structured format, followed by a Follow-up Question Input Box for conversational interaction.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and professionalism.
- Background color: Light gray (#F0F2F5) to provide a clean, uncluttered backdrop.
- Accent color: Subtle purple (#7E57C2) to highlight key actions and elements.
- Body text: 'Inter' (sans-serif) for a modern and readable interface.
- Headline text: 'Space Grotesk' (sans-serif) for impactful headings; use 'Inter' for body text.
- Use clean, minimalist icons from a consistent set (e.g., Material Design Icons).
- Ensure a clear visual hierarchy, using white space effectively to avoid clutter.
- Implement smooth transitions and subtle animations on input submission and data loading for enhanced UX.
