# Legacy Artifacts

This folder keeps old workspace and deployment artifacts that are useful as migration history but are not part of the current local-first Anaya runtime.

## Files

- `firebase-apphosting.yaml`: old Firebase App Hosting configuration from the hosted prototype era.
- `idx-dev.nix`: old Firebase Studio / IDX workspace setup.
- `vscode-idx-settings.json`: old IDX-related VS Code settings.

The current app no longer uses Firebase, Firestore, Firebase App Hosting, or IDX-specific development assumptions.
